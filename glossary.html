Network roles

pNode provider node
The storage machine that holds Xandeum data and earns STOINC. Anyone can run one on a small server; see our how to run a pNode guide. Rewards scale with the storage you commit, your stake, and your boost factor.
vNode validator node
A validator node that supervises the network and coordinates proofs and reward settlement. Different role from the storage-providing pNodes.

Storage concepts

Bucket
A Xandeum storage container that smart contracts read from and write to, like a file in a file system. Buckets are how a Solana program addresses its stored data.
sedApp storage-enabled dApp
A Solana app that uses Xandeum to store and access large or mutable data that would be too expensive to keep in normal Solana accounts (decentralized social, media, big app state).
peek / poke / prove
Xandeum's core primitives: peek reads stored data, poke writes it, and prove verifies it cryptographically. Together they give Solana programs random-access, contract-native storage.
Atlas
The network's coordinator. In the current alpha it is run centrally by the Foundation, with a stated plan to decentralize it to pNodes in beta.
Pulse
The roughly 15-second interval at which Xandeum finalizes a Merkle root of stored state, before anchoring it each Yuga to Solana and then Bitcoin for tamper-evidence.
pRPC
The pNode RPC interface (port 6000) used to query a node for stats and roster data. Foundation guidance is to keep it private (localhost) unless you have a reason to expose it.

Rewards & staking

STOINC Storage Income
Xandeum's system that turns real storage usage into wSOL fees paid to pNode operators, settled each Yuga. The large majority of each pool goes to operators. Estimate yours with the rewards calculator.
Yuga
Xandeum's storage epoch, about 2.8 days. Rewards are tallied and settled once per Yuga. Full explainer: what is a Yuga.
Boost factor
A multiplier on a pNode's rewards. It combines your purchase-era boost with up to two NFT boosts, which multiply together (for example 1.2× × 1.5× = 1.8×). No NFT means 1.0×.
XFDP Xandeum Foundation Delegation Program
On qualifying nodes, the Foundation can match eligible community delegation 1:1, adding backing stake to your node. It increases backing; it is not a guarantee of doubled rewards.
xandSOL
Xandeum's liquid staking token, letting holders stake while keeping a tradable, liquid position.
XAND
Xandeum's token on Solana, used for staking to pNodes and governance. Mint: XANDuUoVoUqniKkpcKhrxmvYJybpJvUxJLr21Gaj3Hx.
