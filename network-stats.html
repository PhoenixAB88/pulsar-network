About these numbers

pNodes and active now come from Xandeum's official mainnet credits feed; a node counts as active if it has reported recently. XAND staked and the current Yuga are decoded straight from the on-chain staking program on Solana. Network storage is summed live from each pNode's gossip-reported committed storage. For the full node-by-node view (uptime, health, location, per-node stake and storage), open the live dashboard.

Network storage is the total capacity committed across every pNode, read live from the network's gossip layer (each node reports its committed and used storage). Used is near zero today because the network's storage is only just starting to be consumed, which is also why STOINC payouts are still small.

FAQ

How many Xandeum pNodes are there?

The live count is shown above, read from the official mainnet feed on each load, plus how many are active right now.

What is the current Yuga?

Shown live above, read from the on-chain staking program. A Yuga is the storage epoch (~2.8 days) after which STOINC rewards settle.

Where do the numbers come from?

Node counts from the official mainnet credits feed; stake and Yuga decoded on-chain via the Xandeum staking program. See the Trust page for how we stay transparent.

Sources: official Xandeum mainnet credits feed · on-chain Xandeum staking program (Solana) · Xandeum.
